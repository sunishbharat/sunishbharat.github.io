AtlasMind: Production AI Assistant

A live natural language to JQL (Jira Query Language) assistant using RAG (Retrieval-Augmented Generation) and pgvector. It features a two-stage router that bypasses the JQL parser for general inquiries. It contains a self-healing loop: when a query fails validation, the error is fed back to the LLM for automatic query correction (up to 4 retry cycles). Operates with 5 runtime-switchable LLM backends.
